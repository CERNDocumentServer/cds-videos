About

The CERN Document Server (CDS) is the institutional repository that provides acquisition, search and collaborative tools to manage collections of documents produced at CERN. The collections include HEP documents, Multimedia documents, Bulletins, Administrative (Directorate, HR, Finance, etc.) documents, EU Project documents (OpenAire, EMi, etc.), Library and Project document types.

CDS is based on Invenio Software.

CDS videos is the first collection on the new Invenio 3 release. All CDS collections will progressively migrate to Invenio 3.
