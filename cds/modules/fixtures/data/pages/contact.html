Help & Contact

Questions related to content

The content submitted to the CDS Videos platform is uploaded by CERN users. The CDS service is only providing technical support for the platform.

If you have questions related to the content (for example how to find content or specific videos), please contact:

  • audiovisual.productions.service@cern.ch for questions related to photos.
  • https://copyright.web.cern.ch/contact for questions related to copyright, conditions of use, etc.
  • helpdesk@cern.ch for general help.

Technical Support

For any question related to the technical issues with the website (error, bug, feature not working, etc...), please contact CDS support. In your message, please provide:

  • the URL where the issue has been found, with the error message and code if available
  • a description of the issue with any relevant information on how to reproduce it
  • if possible and useful, a screenshot
